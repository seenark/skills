# Course Roadmap

คอร์สนี้พาเรียน `octocat/Hello-World` แบบ Complete Guide สำหรับผู้เริ่มต้น โดยใช้ repository เล็กมากเพื่อทำความเข้าใจพื้นฐานที่ใช้ได้กับ project จริงทุกขนาด

## Modules

1. Introduction and Repository Reading
2. README-Driven Project Communication
3. Final Project Challenge

## Learning Flow

เริ่มจากปัญหาว่าทำไม repository ต้องอธิบายตัวเองได้ จากนั้นเรียน mental model ของ README แล้วปิดท้ายด้วยการปรับ README ให้เหมือนงานจริงมากขึ้น
